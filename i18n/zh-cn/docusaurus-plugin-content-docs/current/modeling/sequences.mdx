# Sequences

> [!NOTE]
序列是一项通常仅由关系数据库支持的功能。 如果使用非关系数据库（如 Cosmos），请查看有关生成唯一值的数据库文档。
> 

序列在数据库中生成唯一的顺序数值。 序列不与特定表关联，可以设置多个表以从同一序列提取值。
